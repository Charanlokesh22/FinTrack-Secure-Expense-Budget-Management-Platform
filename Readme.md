FinTrack is a full-stack web application that allows users to track expenses, categorize them, and visualize budgets.  
It includes secure authentication, fast analytics using Redis caching, and a modular backend design.


 Features

- User registration and login with JWT authentication
- Add, edit, and view expenses
- Categorize expenses (Food, Rent, Utilities, etc.)
- Monthly budget tracking
- Dashboard showing spending trends
- Redis caching for frequently accessed analytics


 Tech Stack

- Frontend: React.js
- Backend: FastAPI
- Database: MongoDB
- Caching: Redis
- Authentication: JWT
- API Style: REST
- Tools: Git, Postman
